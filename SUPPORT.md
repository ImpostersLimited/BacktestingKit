# Support

## Getting help

- Read `README.md` and docs in `docs/` first.
- For usage questions, open a GitHub discussion or issue with:
  - Your Swift/Xcode versions.
  - Minimal reproducible example.
  - Expected behavior and actual behavior.

## Bug reports

Please use the bug report issue template and include:

- Steps to reproduce.
- Input data shape (or anonymized sample).
- Error output and stack traces.
- Whether behavior differs from the JavaScript parity baseline.

## Security

For security-sensitive issues, do not open public issues.
Use the private reporting path in `SECURITY.md`.
