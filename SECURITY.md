# Security Policy

If you discover a security issue, please do not open a public issue with exploit details.

Report privately to the project maintainers with:

- Affected area and impact.
- Reproduction steps.
- Suggested mitigation (if known).

We will acknowledge receipt and work on a fix timeline.

## Scope

This policy applies to:

- Core simulation engine code.
- Data ingestion and provider interfaces.
- Build and release automation in this repository.

## Response targets

- Acknowledgement: within 3 business days.
- Initial triage: within 7 business days.
- Fix timeline: communicated after triage based on severity.
